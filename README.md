# ⏱ Stopwatch Web Application
A modern stopwatch web app built using HTML, CSS, and JavaScript.

##Features
- Millisecond precision timing
- Circular progress animation
- Glassmorphism UI
- Vibrant animated background
- Start, Pause, Reset, and Lap functionality

##Tech Stack
- HTML
- CSS
- JavaScript

##Project Structure
- index.html
- css/style.css
- js/script.js
